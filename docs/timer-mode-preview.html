互動規格（提案，待你確認）

  • 單位：只支援分鐘；內部仍以秒倒數（沿用 usePomodoro 的 rAF + Date.now delta）。
  • + / − 設定：每次 ±1 分鐘（可改 ±5）。範圍建議 1–90 分，初始 25 分。
  • 顯示切換：未開始 = 步進器（− 25 +）；按「開始」後步進器收起、顯示 MM:SS 倒數。
  • 鍵盤：沿用 vim 風格焦點移動（h/l、方向鍵）；新增 + / = 加分鐘、- 減分鐘。p 開始/暫停、r 重置（二次確認）不變。
  • 結束行為：時間到發系統通知（沿用現有 sendNotification + c/Esc 關閉）；不自動進入下一階段（與番茄鐘循環不同）。
  • streak（issue #9 原案）：時間到後把 分鐘 / 25 累加到 streak，取小數 2 位（例 12.5 分 = +0.5）。待確認是否本次一起做。
  • 圖片：先沿用 capybara-focus.png；若要專屬圖再加 capybara-timer.png。
  • 模式切換：可比照 Cmd+L 佈局切換，加一個番茄鐘 ⇄ 計時器的切換鍵（待定）。

決策點：(1) 步進間距 ±1 還是 ±5？(2) 分鐘上下限（建議 1–90）？(3) 琥珀色採用或微調？ (4) streak 規則（#9 原案）本次是否一起做？(5) 番茄鐘 ⇄ 計時器要用什麼方式切換？ 確認後我再進實作。
